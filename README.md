# Operating-Systems-NTUA
Series of exercises, for the OS course at [ECE NTUA](https://www.ece.ntua.gr/en) 2020-2021 on operating system fundamentals.

- Processes and Threading
- Inter-process communication
- Synchronization
- Virtual Memory
